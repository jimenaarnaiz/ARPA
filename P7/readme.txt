REALIZACIÓN PRÁCTICA 7

Como aplicación de los tipos de datos derivados vamos plantear una situación con tres procesos. El
proceso 0 inicializará una matriz de tamaño NxN con números aleatorios y los procesos 1 y 2
inicializarán sendas matrices con valores nulos. Posteriormente el proceso 0 enviará al proceso 1 la
matriz triangular superior y al proceso 2 la matriz triangular inferior. Cada uno de los
procesos receptores mostrará su matriz antes y después de recibir los elementos.
